# Bible

The entire King James Version (KJV) Bible for Bangle.js.

## Features
- Complete KJV text (Old and New Testament).
- Dictionary-based compression for space efficiency (~3.9MB total).
- Interactive Book and Chapter selection.
- Smooth scrolling reader.
- Case-preserving text.

## How to use
- Use the menu to pick a Book and then a Chapter.
- Drag up/down on the screen to scroll through the text.
- Press the back button (or swipe left) to return to the selection menus.
