# Labyrinth Marble Run

A simple marble run game controlled by the Bangle.js accelerometer.

## Gameplay
Tilt your watch to control the red marble. Navigate through the maze and reach the green square to progress to the next level. 
Avoid hitting walls too hard, or you might bounce back!

## Controls
- **Tilt Watch**: Moves the marble based on gravity.
- **Button**: Exits the app.

## Features
- Minimalist vector graphics
- Physics engine with collision and bouncing
- Multiple levels
