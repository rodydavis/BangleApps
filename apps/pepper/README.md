# Pepper

Pepper is a virtual pet black cat with yellow eyes that walks around your Bangle.js screen. 

## Interactions
- **Watch**: Pepper will walk around the screen and occasionally stop to rest.
- **Touch**: Tap the screen to interact with Pepper. Pepper will show a heart to indicate she likes it!

## Features
- Cute vector graphics.
- Random walking and idling behavior.
- Touch interaction.
